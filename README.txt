# VR Dogecoin Shooter

An immersive virtual reality shooting game where you interact with Dogecoins in a unique way!

## How to Play
- Pinch your right hand to shoot Dogecoins
- Pinch your left hand to attract Dogecoins

## Device Compatibility
Tested on Quest 3

## Installation
1. Download the APK file from the release section
2. Upload it to your VR headset using Sidequest or similar tools

## Features
- No internet connection required
- Ad-free experience
- No in-game payments

Dive into the world of Dogecoin in VR - shoot, attract, and immerse yourself in this simple yet engaging game!
